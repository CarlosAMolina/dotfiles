## Introduction

Proyect to configure Polybar.

## Steps

Install Polybar.

Create the folder to save the configuration:

```bash
mkdir -p ~/.config/polybar/
```

Copy the `config` file in the previous folder.

Run polybar:

```bash
polybar bar1
```
